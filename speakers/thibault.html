Thibault Imbert gerencia um grupo de produtos para plataforma web focado em gráficos, JavaScript e Adobe Scout, a próxima geração de ferramentas de profiling para tecnologias web.

Thibault lançou em 2008 um free e-book sobre Action Script 3 chamado Pratique d'ActionScript 3. Em 2010, Thibault liderou o Stage3D, que revolucionou os gráficos em Flash, dando muito mais poder para games como FarmVille 2, Angry Birds ou Kings Road no Facebook. Ele recentemente escreveu um livro chamado "Introducing Starling", sobre o framework Starling, liderado por ele para prover aceleração 2D via GPU utilizando Stage3D.

Liderando o time Scout da Adobe, tem o objetivo de mudar a forma como desenvolvedores criam profiles para conteúdos web.

Além disso, escreveu um emulador Intel 8080 em ActionScript, e o traduziu para JavaScript.
